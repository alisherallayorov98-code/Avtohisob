# Avtohisob Roadmap

Bu hujjat loyihani `7 bosqich` bo'yicha oxirigacha olib borish uchun tayanch yo'l xaritasidir.

## 1. Foundation

- Domen modelini barqarorlashtirish
- Demo ma'lumotlar va migrate/seed tayanchini ushlab turish
- API va UI kontraktini tozalash

## 2. Landing va Positioning

- Mahsulot qiymatini aniq ko'rsatish
- Login va onboarding oqimini soddalashtirish
- Modul yo'nalishlarini ko'rinarli qilish

## 3. Core Product UI

- Dashboard, servis, ombor, hisobot va sozlamalarni mukammallashtirish
- Global qidiruv, filterlar, empty/loading holatlar
- Rollar bo'yicha aniq UI tajribasi

## 4. Business Logic

- Servis muddatlari va qoldiq nazoratini kuchaytirish
- Audit, arxivlash va tiklash oqimlari
- KPI va tahliliy ko'rsatkichlarni kengaytirish

## 5. Backend va Data

- Endpointlar barqarorligi
- Validatsiya va xatolik xabarlarini yaxshilash
- Health/status va operatsion tayanch

## 6. Testing va Polishing

- Smoke testlar
- Responsive polishing va UI matnlarni tozalash
- Nozik buglarni yopish

## 7. Release va Production

- README va deploy yo'riqnomasi
- Production checklist
- Monitoring uchun start nuqtalari

## Joriy fokus

Hozirgi iteratsiyada quyidagilar bajarildi:

- roadmap va hujjatlashtirish
- global qidiruv
- dashboard uchun operatsion snapshot
- report filterlarini kengaytirish
- health endpoint va smoke tekshiruvlar
- loading indicator va UX polishing
- report uchun tezkor sana presetlari va oylik kesim
- release checklist va lokal test skriptlari (`check:syntax`, `smoke`)
